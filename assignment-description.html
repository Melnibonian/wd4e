Assignment Description

A few words about my work

Home Page

On the Home page we can first see a full screen custom "carousel". I have used some CSS effects and animations to add some motion and styling. Each instance features a parallax effect using custom vanilla JavaScript. Pressing the arrows changes the background color and animates the image, using custom jQuery.

Then we have a section featuring my project categories using a responsive grid. I have added custom CSS to enable some smooth hovering effects; Clicking on each block redirects you on the respective category;

About Me

On the About Me page I have included a brief bio. Then I use a responsive grid to list my web development skills, using icons that feature some hovering CSS effects. After that I present a short list of my web development training that is followed by a responsive grid, using custom CSS hovering effects, where each block redirects you to the respective certificate (actual certificates will be included in my real portfolio later).

For the certificates I have used a portfolio sorting jQuery plugin, where you can select the category of the certifications you want to view.

Contact

On the Contact page I have created 4 responsive stylized blocks.

  • The first one includes some placeholder contact information.
  • The second one a contact form, which validates user input, where I used custom vanilla Javascript (Github doesn't offer server side programming, so the contact form doesn't actually send a message. Functionality will be inlcuded in my real portfolio later).
  • The third block features an iframe google map.
  • The last one includes some placeholder CSS styled social media links using icons that are validated for accessibility using ARIA - Accessible Rich Internet Applications.

My Projects

Front End Web Design

On the Front End Web Design page I have created a responsive grid, with two items on each row (one in mobile view). Each block represents (and redirects to) a project and consists of two boxes. The first box shows a screenshot of the project and the second box a title and short description. I have used custom css effects and specifically on the description box the mix-blend-mode: difference CSS rule, that dynamically and responsively produces two areas with different background and text color.

You can enter the page of each project and check it, I use CSS and javaScript to produce some visually captivating effects. Enjoy.

Validation

Each page is validated for HTML, CSS & accessibility. I have included on the footer of every page 3 banners, each redirecting you to the respective validation tool of that page.
